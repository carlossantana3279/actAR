# To Run/Test the frontend
1. <a href="https://nodejs.org/en/"> **Install NodeJs and npm** </a>
2. **Install Bower and grunt**
	* ```npm install -g bower grunt-cli```
3. **Run server**
	```
	cd FrontEnd
	grunt server
	```
